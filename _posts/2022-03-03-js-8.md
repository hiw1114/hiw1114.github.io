○boolean
불리언(또는 불린)은 숫자, 문자열과 같은 데이터 타입중 하나이다. true와 false, 단 두 가지 값만 존재한다. 참과 거짓 여부를 나타내기 위헤 사용하는 데이터이다.

let value;
value = true; // 소문자로 써야 합니다.
value = false; // 참, 거짓 모두 마찬가지
value = False; // 이거 안되고
value = True; // 이것도 안 돼요

typeof
데이터 타입을 체크해주는 연산자.
